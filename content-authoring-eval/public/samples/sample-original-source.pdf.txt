## How PDFs Are Used

The content evaluation agent:
1. Fetches PDF using `fetch()`
2. Extracts text using `unpdf` library
3. Compares extracted text with migrated web page content
4. Calculates similarity score using sentence-level diff

## Generating Test PDFs

To create your own test PDFs:

```bash
# Using pandoc (if installed)
pandoc sample-original-source.html -o sample-original-source.pdf

# Or upload to Azure Blob Storage
az storage blob upload \
  --account-name dalivemcprg94e3 \
  --container-name contentsource \
  --name pdf-test/sample.pdf \
  --file ./sample.pdf
```
